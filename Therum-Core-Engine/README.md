# THERUM — Flagship Hybrid Synth Platform

THERUM is the flagship **sound-creation** product in the suite.

It is intended to:
- compete with top-tier hybrid synths
- exceed them in clarity, route visibility, and learnability
- stay in the **creation lane** so it does not conflict with:
  - FreeEQ8 (precision shaping)
  - BassMaid (low-end control)
  - GlueMaid (cohesion)
  - SpaceMaid (spatial finishing)
  - MixMaid (bounded correction)
  - AETHER (ethereal choir atmosphere)
  - WhisperGate (ritual experimental texture)

## Target role
Create the sound at the source.

## Core feature target
- Osc A / B / Sub / Noise
- Wavetable / Sample / Granular / Spectral / VA
- 10 LFO, 6 ENV, 8 MACRO
- Dual filters
- Full modulation matrix
- Patch-level FX rack
- Diagnostics and CPU visibility
- Dark-neon GUI
- Reactive boxer mascot panel ("Punch Reactor")
