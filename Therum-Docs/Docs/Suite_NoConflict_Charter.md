# Suite No-Conflict Charter

## THERUM
Flagship synth creation platform.

## FreeEQ8
Precision shaping.

## BassMaid
Low-end control.

## GlueMaid
Cohesion / bus glue.

## SpaceMaid
Spatial finishing.

## MixMaid
Bounded mix correction.

## AETHER
Ethereal choir atmosphere specialist.

## WhisperGate
Ritual / experimental texture explorer.

## Key rule
THERUM may include internal EQ / comp / reverb / delay / distortion,
but only as **patch-completion tools**. It must never be marketed as replacing
the dedicated suite processors.
