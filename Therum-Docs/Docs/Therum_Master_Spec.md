# THERUM MASTER SPEC v1

## Product thesis
THERUM should match flagship synth breadth while exceeding clarity and performance discipline.

## Pillars
1. Clarity over chaos
2. Performance discipline
3. Explorability
4. Patch explainability
5. Premium dark-neon presentation

## Feature boundary
THERUM owns:
- sound creation
- synthesis depth
- modulation
- patch FX
- patch diagnostics

THERUM does NOT own:
- final mix correction
- dedicated surgical EQ workflow
- dedicated bus glue identity
- dedicated spatial finishing identity
- specialized choir-atmosphere branding
- ritual experimental branding
