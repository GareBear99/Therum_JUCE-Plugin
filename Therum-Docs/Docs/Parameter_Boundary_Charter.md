# Parameter Boundary Charter

THERUM parameters should be organized in families:

## Global
- masterGain
- polyphony
- qualityMode

## Oscillator A / B
- mode
- wtPos
- warp
- blend
- unison
- detune
- level
- pan

## Sub / Noise
- level
- shape / color
- tracking
- oneShot (noise)

## Filters
- routing
- type
- cutoff
- resonance
- drive
- keytrack
- mix

## Modulation
- lfo*
- env*
- macro*
- random*

## FX
- distortion*
- chorus*
- delay*
- reverb*
- eq*
- comp*
- limiter*

## Reserved roles
Patch-completion parameters live here only for sound design.
Track/mix/master processing roles remain reserved for the dedicated suite plugins.
